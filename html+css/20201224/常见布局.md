# 常见布局
## 三段式
header: 页头，多用于以下用途
- 展示网站名称
- 导航栏
- 搜索框
- 用户信息

content(main): 内容区域，用于展示页面的具体内容

footer: 页脚（脚注），多用于以下用途：
- 标注证书版权等信息
- 合作伙伴
- 功能性链接（如：论坛地址，文档地址，问答通道等）

## 后台布局方式
基本和三段式布局相同，不同点：
- 拥有侧边栏，用于导航
- header当中导航很少，大部分导航挪到了侧边栏中
- footer中，内容较少，基本都是标注版权信息（也有维护人员信息）
- 导航不再进行页面跳转，而是在页面content中显示新的内容
